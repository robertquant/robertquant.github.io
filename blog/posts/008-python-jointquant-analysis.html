二、核心策略深度解析

1. ETF动量轮动策略（ETF_rotation_v1.py）

策略逻辑亮点

这个策略采用了加权线性回归计算动量，是一个非常有意思的设计：

# 动量评分计算 综合评分 = 年化收益率 × R² 其中： - 年化收益率：基于加权线性回归的斜率 - R²：判定系数，衡量趋势稳定性 - 权重：线性递增（近期数据权重更高）

✅ 优势：

  • R²过滤可以剔除"不稳定的高收益"，避免假突破
  • 加权线性回归让近期价格对动量影响更大
  • 安全区间过滤（0 < score ≤ 5）控制极端值

⚠️ 可优化：

  • 建议加入波动率过滤（ATR或标准差）
  • 可考虑加入大盘趋势判断（如沪深300是否在20日均线上方）
  • 止损机制（如-8%硬止损）尚未体现

2. 小市值轮动策略（small_cap.py）

多因子过滤设计

这个策略不只是简单选小市值，而是加入了一系列质量因子过滤：

过滤条件： 1. ST股票过滤 2. 停牌过滤 3. PE > 0（盈利公司） 4. 上市满365天 5. 负债率 < 70% 6. ROE > 5% 7. 经营现金流 > 0

✅ 优势：

  • 多重过滤有效避免了"垃圾股"和财务风险
  • ROE和现金流要求确保公司质量
  • 5日调仓频率平衡了交易成本和策略灵敏度

⚠️ 风险提示：

  • 小市值策略在流动性紧张时可能面临冲击成本
  • 建议加入流动性过滤（日均成交额 > 阈值）
  • 2024年以来小市值因子波动加大，需警惕风格切换

3. AH股溢价套利策略（ah_premium_strategy.py）

跨市场套利逻辑

利用A股和H股的价差进行套利，这是一个典型的均值回归策略：

套利逻辑： 1. 当 AH溢价率 较低时（A股相对H股折价）→ 买入A股 2. 当 AH溢价率 较高时（A股相对H股溢价）→ 卖出平仓 核心假设：同一公司的A股和H股长期价格应趋于一致

✅ 优势：

  • 逻辑清晰，基于一价定律
  • 与A股大盘相关性较低，分散风险
  • 聚宽内置数据，数据源稳定

⚠️ 风险点：

  • AH溢价可能长期不收敛（受汇率、流动性、投资者结构影响）
  • 港股通有额度限制和T+0规则差异
  • 汇率波动会影响实际收益

4. RSRS择时策略（rsrs_value_strategy.py）

阻力支撑相对强度

RSRS是一个比较少见的技术面择时指标，通过计算最高价和最低价的线性回归斜率来判断趋势强度：

RSRS计算： 1. 取N日最高价和最低价 2. 对最高价和最低价做线性回归 3. 斜率即为RSRS值 4. 斜率 > 阈值 → 趋势向上，开仓 5. 斜率 < 阈值 → 趋势向下，空仓

✅ 优势：

  • RSRS比单纯的价格突破更能反映趋势强度
  • 与选股结合，实现了"选股+择时"双保险
  • 空仓机制在熊市能有效控制回撤

三、代码质量评估

✅ 优点

  1. 文档完善 - README.md详细说明了每个策略的逻辑、参数和风险
  2. 结构清晰 - 策略按类型分类，命名规范
  3. 风险控制意识 - 大部分策略有过滤条件（ST、停牌、负债率等）
  4. 交易成本考虑 - 设置了合理的佣金和滑点
  5. 多策略覆盖 - 轮动、选股、套利、择时多维度布局

⚠️ 可改进点

  1. 缺乏止损机制 - 大部分策略没有硬止损或移动止损
  2. 回测报告缺失 - 没有附带历史回测结果（收益率、回撤、夏普等）
  3. 参数优化不足 - 参数设置偏经验值，缺乏敏感性分析
  4. 实盘对接 - 注释掉的 redistrade 导入说明有实盘计划，但未完整实现
  5. 测试覆盖 - 没有单元测试，策略正确性依赖手动验证

四、策略组合建议

基于这些策略，建议构建一个多策略组合：

仓位分配 策略 作用
30% ETF动量轮动 获取大盘beta收益
25% 小市值轮动 获取小盘alpha收益
20% 低估值选股 价值因子暴露
15% AH溢价套利 低风险套利收益

组合优势：
• ETF轮动提供大盘敞口，避免踏空
• 小市值+低估值双因子选股，分散风格风险
• AH套利提供与A股低相关的收益
• 整体回撤可控（各策略相关性较低）
